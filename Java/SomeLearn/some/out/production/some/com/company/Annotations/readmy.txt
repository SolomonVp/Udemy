                        - - - A N N O T A T I O N - - -
         Аннтоация - это специальные комментарии\метки\метаданные, которые нужны для передачи определенной информации.

         @Target показывает область кода, к которой Аннтоация может быть применима. Самые распространенные области кода:
                            - TYPE - class, interface, enum;
                            - FIELD - поля класса;
                            - METHOD - метод класса;
                            - PARAMETER - праметры метода или конструктора;

         @Retention описывает жизненный цикл АннотацииЖ
                            - SOURCE - Аннотация видна в source коде, отбрасывается компилятором и уже в byte коде не видна.
                            - CLASS - Аннтоация видна в byte коде, отбрасывается JVM во время выполнения программы.
                            - RUNTIME - Аннотация видна во время выполнения программы.
