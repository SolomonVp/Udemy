            Регулярные выражения необходимы для создания шаблонов, с помощью которыхпроизводят такие операции как поиск,
            сравнение, замена.

            Регулярные выражения - это совокупность символов, некоторые из которых являются специальными метасимволами,
            т.е. обладают каким-то функционалом.

            ------------------------------------------------------------------------------------------------------------

            String s1 = "ABCD ABCE ABCFABCGABCH";
            Pattern pattern1 = Pattern.compile("ABCD");
            Matcher matcher = pattern1.matcher(s1);

            Pattern - шаблон на примере которого мы будем искать в объекте нужные нам выражения.
            Matcher - находит соответствия. Мы наш шаблон(Pattern) сверяем с субъектом, в котором нужно найти наш объект
            Pattern.

            ------------------------------------------------------------------------------------------------------------
            abc                 - соответствует последовательно идущим abc.
            [abc]               - соответствует a, b, или c.
            [^abc]              - НЕ соответствует a, b, или c.
            [d-j]               - одна из букв от d до j.
            AB[C-F]OP           - найдет выражение AB[любая буква в диапазоне C - F]OP
            [3-8]               - цифра от 3 до 8.
            [D-Fd-j3-8]         - одна ил букв из обоих диапазонов или цифра от 3 до 8.
            cd(a|b)             - a или b.
