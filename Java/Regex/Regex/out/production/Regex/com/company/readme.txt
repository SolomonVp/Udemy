            Регулярные выражения необходимы для создания шаблонов, с помощью которыхпроизводят такие операции как поиск,
            сравнение, замена.

            Регулярные выражения - это совокупность символов, некоторые из которых являются специальными метасимволами,
            т.е. обладают каким-то функционалом.

            ------------------------------------------------------------------------------------------------------------

            String s1 = "ABCD ABCE ABCFABCGABCH";
            Pattern pattern1 = Pattern.compile("ABCD");
            Matcher matcher = pattern1.matcher(s1);

            Pattern - шаблон на примере которого мы будем искать в объекте нужные нам выражения.
            Matcher - находит соответствия. Наш шаблон(Pattern) сверяем с объектом, в котором нужно найти Pattern.

            ------------------------------------------------------------------------------------------------------------
            abc                 - Соответствует последовательно идущим abc.
            [abc]               - Соответствует a, b, или c.
            [^abc]              - НЕ соответствует a, b, или c.
            [d-j]               - Соответствует одна из букв от d до j.
            AB[C-F]OP           - найдет выражение AB[любая буква в диапазоне C - F]OP
            [3-8]               - Соответствует цифра от 3 до 8.
            [D-Fd-j3-8]         - Соответствует одна из букв из обоих диапазонов или цифра от 3 до 8.
            cd(a|b)             - Соответствует a или b.
            .                   - Соответствует одному любому символу. Исключение: символ новой строки.
                                  Чем больше точек, там больше символов.
            ^выражение          - соответствует выражению в начале строки.
            выражение$          - Соответствует выражению в конце строки.
            \d                  - Соответствует одной цифре. (Необходимо отражать через "\").
            \D                  - Соответствует одной НЕ цифре. (Необходимо отражать через "\").
            \w                  - Соответствует одной букве, цифре или "_". (Необходимо отражать через "\").
            \W                  - Соответствует одному символу, который НЕ буква, НЕ цифра и НЕ "_". (Необходимо отражать через "\").
            \s                  - Соответствует пробельному символу. (Необходимо отражать через "\").
            \S                  - Соответствует НЕ пробельному символу. (Необходимо отражать через "\").
            \A                  - Соответствует выражению в начале String-a.
            \Z                  - Соответствует выражению в конце String-a.
            \b                  - Соответствует границе слова или числа.
            \B                  - Соответствует границе не слова и не числа.

            выражение?          - Соответствует 0 или 1 повторению.
            выражение*          - Соответствует 0 или большему количеству повторений.
            выражение+          - Соответствует 1 или больше количеству птовторений.
            выражение{n}        - Соответствует количеству повторений "n". n - это число.
            выражение{m, n}     - Соответствует количеству повторений от "m" до "n". m и n - это число.
            выражение{n,}       - Соответствует n или большему количеству повторений. n - это число.

            public boolean matches (String regex) - возвращает boolean(true\false), сверяет String с регулярным выражением.
            publicString [] split (String regex) - делит String на несколько частей. Принцип деоения указываем в regex.

            %[flags][width][.precision]datatype_specifier

            flags:

            "-"                 - выравнивание по левому краю
            "0"                 - добавление нулей перед числом
            ","                 - разделитель разрядов в числах

            DT specifiers:

            "b"                 - boolean
            "c"                 - character
            "s"                 - String
            "d"                 - целое число
            "f"                 - десятое число
